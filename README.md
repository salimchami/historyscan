# History Scan Project

## Description

This project is a Java application built with Spring Boot and Gradle. It involves features for codebase Git analysis.

## Getting Started

### Dependencies

- Java
- Spring Boot
- Gradle

### Install and start

Start the spring boot application on the default port.

```
    ./gradlew build
    ./gradlew bootRun
```
